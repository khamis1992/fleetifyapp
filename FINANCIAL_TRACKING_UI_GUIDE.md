## 🎯 Button States

### **Export Button:**
```
Enabled:  [📊 تصدير Excel]  (Customer selected, has receipts)
Disabled: [📊 تصدير Excel]  (No customer or no receipts)
Loading:  [⏳ جاري التصدير...] (During export)
Success:  [✅ تم التصدير]     (After successful export)
```

### **Print All Button:**
```
Enabled:  [🖨️ طباعة الكل]  (Customer selected, has receipts)
Disabled: [🖨️ طباعة الكل]  (No customer or no receipts)
```

## 🎨 Color Coding

### **In UI:**
- 🔵 **Blue** - Export button
- 🟢 **Green** - Success messages
- 🔴 **Red** - Fine amounts
- ⚫ **Gray** - Print buttons (neutral)

### **In Print:**
- 🔴 **Red** - Fine badge (#fee background, #c00 text)
- ⚫ **Black** - Headers (#333)
- 🔵 **Blue** - Total payments
- 🟢 **Green** - Receipt count

## 🎯 User Flow

### **Export Flow:**
```
1. User clicks "تصدير Excel"
   ↓
2. System validates data
   ↓
3. Creates CSV content
   ↓
4. Triggers download
   ↓
5. Shows success toast
   ✅ "تم تصدير البيانات بنجاح"
```

### **Print Receipt Flow:**
```
1. User clicks 🖨️ icon
   ↓
2. System generates HTML
   ↓
3. Opens new window
   ↓
4. Shows print preview
   ↓
5. User clicks "طباعة"
   ↓
6. Browser print dialog opens
   ✅ Receipt prints
```
