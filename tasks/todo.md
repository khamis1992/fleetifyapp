# تحسين تصدير المذكرة الشارحة - Word و PDF

## الهدف
إضافة إمكانية تحميل المذكرة الشارحة بصيغة Word (docx) و PDF في صفحة تجهيز الدعوى.

## المهام

- [x] 1. إضافة مكتبة `docx` لتوليد ملفات Word
- [x] 2. إنشاء دالة `downloadAsPdf` لتحويل HTML إلى PDF
- [x] 3. إنشاء دالة `downloadAsDocx` لتحويل HTML إلى Word
- [x] 4. تحديث صفحة `LawsuitPreparation.tsx` لإضافة أزرار تحميل PDF و Word
- [x] 5. اختبار التحميل والتأكد من جودة المخرجات

## المراجعة

### الملفات المعدلة:
1. **package.json** - تم إضافة مكتبتي `docx` و `file-saver`
2. **src/utils/document-export.ts** - ملف جديد يحتوي على دوال التصدير
3. **src/pages/legal/LawsuitPreparation.tsx** - تم إضافة أزرار PDF و Word

### التغييرات:
- زر **PDF** أحمر اللون لتحميل المذكرة بصيغة PDF
- زر **Word** أزرق اللون لتحميل المذكرة بصيغة DOCX
- الأزرار تظهر فقط بعد توليد المذكرة الشارحة
- اسم الملف يتضمن اسم العميل والتاريخ

### ملاحظات تقنية:
- PDF يُنشأ باستخدام `html2canvas` + `jspdf`
- Word يُنشأ باستخدام مكتبة `docx` مع دعم اللغة العربية (RTL)
