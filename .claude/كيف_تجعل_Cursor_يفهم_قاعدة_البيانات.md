## 🎯 كيفية استخدام النظام

### عند العمل مع قاعدة البيانات:

1. **Cursor سيقرأ تلقائياً** `.cursorrules`
2. **الملف يوجهه** إلى `.claude/DATABASE_SCHEMA_REFERENCE.md`
3. **سيعرف البنية الصحيحة** بدون الحاجة لـ MCP!

---

## 📋 مثال عملي

**قبل:**
```typescript
// Cursor قد يخطئ في التخمين:
await supabase
  .from('journal_entry_lines')
  .insert({
    description: 'test',  // ❌ خطأ - العمود اسمه line_description
    level: 1              // ❌ خطأ - لا يوجد عمود level
  });
```

**بعد (مع المرجع):**
```typescript
// Cursor سيعرف الأعمدة الصحيحة:
await supabase
  .from('journal_entry_lines')
  .insert({
    line_description: 'test',  // ✅ صحيح
    line_number: 1              // ✅ صحيح
  });
```
