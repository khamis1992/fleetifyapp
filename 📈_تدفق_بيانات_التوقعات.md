## 🔢 المعادلات المستخدمة

### 1. الإيراد الحالي
```sql
SELECT SUM(total_debit - total_credit) as totalRevenue
FROM journal_entries
WHERE account_code LIKE '41%' -- حسابات الإيرادات
  AND company_id = 'current_company'
  AND entry_date >= 'start_of_month'
  AND entry_date <= 'end_of_month'
```

### 2. الإيراد المتوقع
```typescript
const forecastedRevenue = currentRevenue * 1.18
```

### 3. نسبة النمو
```typescript
const growthRate = ((forecastedRevenue - currentRevenue) / currentRevenue) * 100
// النتيجة: +22%
```

### 4. نسبة الإنجاز
```typescript
const currentProgress = (currentRevenue / targetRevenue) * 100
// الشهر الحالي: 78%

const forecastedProgress = (forecastedRevenue / targetRevenue) * 100
// الشهر القادم: 92%
```

## 🎯 نقاط البيانات الرئيسية

### مدخلات النظام:
```
1. الإيراد الحالي → من قاعدة البيانات
2. معامل النمو → ثابت 18%
3. العوامل المؤثرة → محددة مسبقاً
4. دقة التوقع → محددة مسبقاً (85%)
```

### مخرجات النظام:
```
1. الإيراد المتوقع → محسوب
2. نسبة النمو → محسوبة (+22%)
3. نسبة الإنجاز → محسوبة (78% → 92%)
4. العوامل البصرية → معروضة
```

## 📊 مصادر البيانات بالتفصيل

### 1. من جدول `journal_entries`:
```typescript
// الإيرادات (الحسابات تبدأ بـ 41)
SELECT SUM(total_credit) FROM journal_entries 
WHERE account_code LIKE '41%'

// مثال:
// 41101 - إيرادات تأجير سيارات
// 41102 - إيرادات خدمات إضافية
// 41103 - إيرادات غرامات التأخير
```

### 2. من جدول `payments`:
```typescript
// المدفوعات المسجلة
SELECT SUM(amount) FROM payments
WHERE status = 'completed'
  AND payment_date BETWEEN start_date AND end_date
```

### 3. من جدول `contracts`:
```typescript
// العقود النشطة
SELECT COUNT(*), SUM(total_amount) FROM contracts
WHERE status = 'active'
  AND company_id = 'current_company'
```

### 4. من جدول `invoices`:
```typescript
// الفواتير الصادرة
SELECT SUM(total_amount) FROM invoices
WHERE status = 'paid'
  AND issue_date BETWEEN start_date AND end_date
```
