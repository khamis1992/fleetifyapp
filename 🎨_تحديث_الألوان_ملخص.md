## 🎨 نظام الألوان المعتمد

### الأساسي (Primary) - الأحمر
```css
/* Light backgrounds */
bg-red-50
bg-red-100

/* Medium */
bg-red-400
bg-red-500

/* Dark - الأكثر استخداماً */
bg-red-600
bg-red-700
```

### المميز (Accent) - البرتقالي
```css
/* Light backgrounds */
bg-orange-50
bg-orange-100

/* Medium */
bg-orange-400
bg-orange-500

/* Dark */
bg-orange-600
bg-orange-700
```

### Gradient Combinations
```css
/* للخلفيات الفاتحة */
bg-gradient-to-r from-red-50 to-orange-50
bg-gradient-to-br from-orange-50 to-red-50

/* للأزرار */
bg-gradient-to-r from-red-500 to-red-600
bg-gradient-to-r from-orange-500 to-orange-600
bg-gradient-to-r from-red-600 to-orange-600

/* للنصوص (gradient text) */
bg-gradient-to-r from-red-600 to-orange-600 bg-clip-text text-transparent
```
