```css
/* Card Colors */

--card-foreground: النظام الموحد الحالي

/* Border Colors */
--border: شفاف 50%
--border/20: شفاف 20%

/* Gradient Colors */
--blue-500/10: أزرق شفاف للعملاء

--purple-500/10: بنفسجي شفاف للعقود
--emerald-500/10: زمردي شفاف للمدفوعات
```
