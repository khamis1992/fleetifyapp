## 🎨 نظام الألوان النهائي

### الألوان الرئيسية المستخدمة:

#### 🔴 الأحمر (Primary)
- **الاستخدام:** العلامة التجارية، الأزرار الرئيسية، البطاقات المهمة
- **Classes:** `from-red-500 to-red-600`, `text-red-600`, `bg-red-50`
- **مثال:** شريط التقدم الحالي، بطاقة المركبات

#### 🟠 البرتقالي (Accent)
- **الاستخدام:** التمييز، الأزرار الثانوية، التحذيرات
- **Classes:** `from-orange-500 to-orange-600`, `text-orange-600`, `bg-orange-50`
- **مثال:** بطاقة العقود، حالة الصيانة

#### 🔴🟠 Gradients (مزيج أحمر-برتقالي)
- **الاستخدام:** عناصر premium، نصوص مهمة
- **Classes:** `from-red-600 to-orange-600`, `from-orange-600 to-red-600`
- **مثال:** أرقام النمو المتوقع والدقة

#### 🟢 الأخضر (Success)
- **الاستخدام:** النجاح، الحالات الإيجابية، المتاح
- **Classes:** `from-emerald-500 to-emerald-600`, `text-emerald-600`
- **مثال:** النمو الإيجابي (+22%), حالة متاح

#### ⚠️ الأصفر/الكهرماني (Warning)
- **الاستخدام:** التحذيرات (يمكن استبداله بالبرتقالي)
- **Classes:** `from-amber-500 to-yellow-600`
- **مثال:** تنبيهات الصيانة

## 💡 نصائح للاستخدام

```tsx

</p>
```

### Glass Card Premium
بطاقة شفافة فاخرة:
```tsx
<div className="glass-card rounded-3xl p-6 border-t border-red-500/20 relative overflow-hidden">
  {/* Background glow */}
  <div className="absolute -bottom-8 -left-8 w-32 h-32 bg-gradient-to-r from-red-500 to-orange-500 rounded-full blur-2xl opacity-20" />
  
  {/* Content */}
  المحتوى هنا
</div>
```

### Icon with Gradient Background
أيقونة مع خلفية gradient:
```tsx
<div className="p-4 bg-gradient-to-br from-red-500 to-red-600 rounded-2xl shadow-lg hover:scale-110 transition-transform">
  <Icon className="w-8 h-8 text-white" />
</div>
```

### Progress Bar

```tsx

    className="h-full bg-gradient-to-r from-red-500 to-red-600 rounded-full" 
    style={{ width: '78%' }}
  />
</div>
```
