## 🎨 نظام الألوان

### بطاقات الإحصائيات
```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 🔵 أزرق      │  │ 🟢 أخضر      │  │ 🟠 برتقالي    │  │ 🟣 بنفسجي    │
│ إجمالي       │  │ نجاح         │  │ معلق         │  │ عملاء        │
└──────────────┘  └──────────────┘  └──────────────┘  └──────────────┘
```

### الشارات (Badges)
```
🟦 تذكير مبكر        (أزرق فاتح)
⬜ يوم الاستحقاق      (رمادي)
🟥 متأخر             (أحمر)
🟥 تصعيد             (أحمر غامق)
```

### حالات الرسائل
```
✅ تم الإرسال        (أخضر)
⏰ معلق              (برتقالي)
❌ فاشل              (أحمر)
🔄 قيد الإرسال       (أزرق)
```

```
✅ تم التحديث بنجاح!
```

```
💾 جاري التصدير...

```


## ✨ الحركات والتأثيرات

```
🎨 Hover على البطاقة:
   قبل: [         ]
   بعد: [  ⬆️ رفع بسيط + ظل أكبر  ]

🎨 تحميل البيانات:
   ⚪⚪⚪ → ⚫⚪⚪ → ⚪⚫⚪ → ⚪⚪⚫ (نبض)

🎨 ظهور البطاقات:
   1. الأولى تظهر
   2. بعد 50ms الثانية
   3. بعد 100ms الثالثة
   (تأثير Stagger)
```
